# Game of Snap 🎴

A browser version of the card game Snap.

## Tech Stack

- **React 19**
- **TypeScript**
- **Vite**
- **Sass**
- **Font Awesome**
- **Vitest** + **React Testing Library**
- **ESLint**

## Run Locally

You'll need installed [Node.js](https://nodejs.org/)

- Clone the repo
- CD into the directory gameofsnap
- Install the app dependencies by running the command npm install
- Start the app on your local server by running the command npm run dev

## Commands

```bash
npm install
npm run dev
npm test
npm run lint
```
